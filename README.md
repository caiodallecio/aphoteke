## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# all tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# unit tests
$ yarn run test:unit

# all test coverage
$ yarn run test:cov

# e2e test coverage
$ yarn run test:e2e:cov

# unit test coverage
$ yarn run test:unit:cov
```
